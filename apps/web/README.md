# Arx - Planora Hall

A modern, responsive web interface for [Planora](https://github.com/planora-proj/planora),
designed for seamless interaction with the backend API.

This project focuses on usability, performance, and developer productivity

## Tech Stach

- **Framework** - Next.js
- **Language** - TypeScript
- **Styling** - Tailwind css, Shadcn UI

## Project Setup

#### 1. Clone the repository

```bash
    git clone https://github.com/planora-proj/planora
    cd planora/apps/web
```

#### 2. Install dependencies

```bash
    pnpm install
```

#### 3. Start the development server

```bash
    pnpm dev
```

Now visit [http://localhost:3000](http://localhost:3000)
