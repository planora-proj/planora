## What is Planora?

Planora is built to simplify how teams **plan, collaborate, and execute** their projects  
It provides a clean interface, real-time task management, and a collaborative workspace to help you stay focused on what matters.

While still in its **early-access phase**, Planora aims to evolve into a robust ecosystem that combines **task management, scheduling, and communication** into one cohesive experience.

---

## Features

- **Project Planning:** Create, organize, and manage multiple projects with ease.  
- **Task Management:** Assign, track, and update tasks effortlessly.  
- **Team Collaboration:** Manage roles, responsibilities, and progress in real-time.  
- **Scalable Architecture:** Backend powered by [Actix Web](https://actix.rs/) and PostgreSQL for high performance.  
- **Modern UI:** Built with Next.js, Tailwind CSS, and shadcn/ui for a beautiful, accessible experience.  
- **Community-Driven:** Open for discussions, suggestions, and contributions.
